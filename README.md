# BudgetIQ

A modern personal budgeting and finance tracker built with React, TypeScript, and Tailwind CSS.

## Features

- **Dashboard** — Balance overview, income vs expenses, budget progress, savings goals, smart insights
- **Transactions** — Full CRUD with search, filters, and auto-categorization
- **Budget** — Per-category monthly limits with progress bars and over-budget alerts
- **Analytics** — Pie chart, spending trends, monthly comparisons, and insights
- **Goals** — Create savings goals, track progress, add contributions
- **Smart features** — Auto-categorize transactions, budget suggestions, safe-to-spend estimate
- **Dark mode** — Light/dark theme toggle with system preference detection
- **Local storage** — All data persists in the browser (no login required)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

- Vite
